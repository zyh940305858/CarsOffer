# 仿车轮网项目(M站)

## 技术栈
    vue4.X + vue-router + vuex + Vant + Eslint + ES6 + axios

## 技术点
- 数据的重构、去重
- 组件的封装
- 移动端适配
- 全局loading的实现
- vuex的modules数据管理+
- services请求接口管理
- axios的二次封装

## 项目运行
```
git clone git@github.com:zyh940305858/CarsOffer.git

npm install 或 yarn(推荐)

npm run serve
```
