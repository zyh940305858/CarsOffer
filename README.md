# 汽车报价项目(M站)

## 技术栈
    vue + vue-router + vuex + Vant + Eslint + ES6 + axios

## 技术点
- 数据的重构、去重
- 组件的封装
- 移动端适配
- 全局loading的实现
- vuex的modules数据管理+
- services请求接口管理
- axios的二次封装
- better-scroll实现上拉加载 下拉刷新
- Vant组件引用
- CSS3动画
- 防抖、节流优化

## 项目运行
```
git clone git@github.com:zyh940305858/CarsOffer.git

npm install 或 yarn(推荐)

npm run serve
```
